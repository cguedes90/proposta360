Criando sua Primeira Proposta

Vamos começar do básico - criando uma nova proposta no dashboard

Como Começar

  • 1
    No dashboard, clique em "Nova Proposta"
  • 2
    Insira o título da proposta (ex: "Transformação Digital 2024")
  • 3
    Escolha se deseja começar do zero ou usar um template
  • 4
    Clique em "Criar Proposta" para abrir o editor

Dicas Importantes

  • 💡
    Use títulos que despertem interesse e urgência
  • 🎯
    Pense no seu cliente ao escolher o nome
  • 📊
    Templates aceleram o processo em 70%
  • ✨
    Você pode editar tudo depois de criada

Definindo Cores e Identidade Visual

A identidade visual é crucial - ela comunica profissionalismo e confiança

Configurando as Cores no Editor

  • 🎨
    No topo do editor, clique no botão "⚙️ Configurações"
  • 🎯
    Na aba "Estilo", encontre "Paleta de Cores"
  • ✨
    Clique na cor primária para abrir o seletor
  • 🔄
    As cores secundárias se ajustam automaticamente

Psicologia das Cores

  • 💙
    Azul: Confiança, tecnologia, estabilidade
  • 💚
    Verde: Crescimento, sucesso, prosperidade
  • ❤️
    Vermelho: Urgência, paixão, energia
  • 💜
    Roxo: Luxo, criatividade, inovação

Adicionando Textos Impactantes

Aprenda a usar textos que vendem - títulos, descrições e calls-to-action eficazes

Adicionando Textos no Editor

  • 📝
    No painel esquerdo, clique em "📝 Texto"
  • 🖱️
    Arraste para o canvas central (área pontilhada)
  • ✏️
    Duplo-clique no texto para começar a digitar
  • 🎨
    No painel direito, ajuste cor, tamanho e fonte

Textos que Convertem

  • 🎯
    Títulos: Foque no benefício, não no recurso
  • 📈
    Descrições: Use números e dados específicos
  • ⚡
    CTAs: Crie senso de urgência
  • ❤️
    Benefícios: Fale das emoções do cliente

Exemplo: Títulos Que Vendem vs Títulos Comuns

❌ Títulos Comuns

"Desenvolvimento de Website"
"Consultoria em Marketing"
"Serviços de TI"

✅ Títulos Que Vendem

"Website que Converte 3x Mais Clientes"
"Estratégia que Triplicou Vendas em 90 Dias"
"Sistema que Automatiza 80% do Trabalho"

Dominando o Editor Visual

Aprenda a usar nosso editor drag-and-drop para criar propostas profissionais rapidamente

Usando o Editor do Proposta 360º

  • 📦
    No painel esquerdo, vá na aba "Elementos"
  • 🖱️
    Arraste elementos como "Texto", "Imagem", "Botão" para o canvas central
  • ✏️
    Clique duplo em qualquer elemento para editá-lo
  • 🎨
    Use o painel direito para ajustar cores, fontes e espaçamento

Dicas do Editor

  • 🔄
    Ctrl+Z: Desfazer a última ação
  • 👁️
    Preview: Clique no olho para ver como ficará
  • 📱
    Responsivo: Teste em diferentes tamanhos de tela
  • 💾
    Auto-save: Suas alterações são salvas automaticamente
